# [자료구조1] Assignment 14 - 2026.05.17

## 과제
-  깃허브에 등록된 연결 리스트 소스 코드와 공지사항의 ADT를 바탕으로, 사용자가 메뉴를 통해 정수 목록을 자유롭게 추가, 삭제, 검색할 수 있는 프로그램을 작성하시오. 메뉴는 다음과 같다.
 - 모든 연산은 이중 원형 연결 리스트의 특성이 적용된 연산을 활용해 동작해야 한다.
  1. 맨 앞에 추가: 정수를 입력받아 리스트의 가장 앞에 삽입
  2. 맨 뒤에 추가: 정수를 입력받아 리스트의 가장 뒤에 삽입
  3. 특정 위치에 추가: 위치(index)와 정수를 입력받아 해당 위치에 삽입
  4. 특정 위치 삭제: 위치(index)를 입력받아 해당 노드를 삭제하고 값 출력
  5. 전체 목록 출력: 현재 리스트에 저장된 모든 정수를 순서대로 출력
  6. 리스트 크기 확인: 현재 저장된 노드의 총 개수를 출력
  0. 종료: 모든 메모리를 해제(destroy)하고 프로그램을 종료
---

## 내용

### 메뉴 기반 프로그램 구현
반복문을 이용하여 사용자가 원하는 메뉴를 계속 선택할 수 있도록 구현하였다.

```c
while (menu != 0)
```

메뉴 번호 입력에 따라 각 이중 원형 연결 리스트 기능이 실행되도록 구성하였다.

---

### 이중 원형 연결 리스트 생성
`createDLinkedList()` 함수를 이용하여 head 노드를 생성하고,  
빈 리스트에서도 원형 구조가 유지되도록 구현하였다.

```c
temp->head.prev = &(temp->head);
temp->head.next = &(temp->head);
```

head의 prev와 next가 자기 자신을 가리키도록 설정하였다.

---

### 맨 앞 삽입 기능
`insertFirstDLinkedList()` 함수를 이용하여 새로운 노드를 리스트의 가장 앞에 삽입하도록 구현하였다.

```c
nPtr->prev = &(dl->head);
nPtr->next = dl->head.next;
```

새 노드를 head 바로 뒤에 연결하도록 구현하였다.

---

### 맨 뒤 삽입 기능
`insertLastDLinkedList()` 함수를 이용하여 새로운 노드를 리스트의 마지막 위치에 삽입하도록 구현하였다.

```c
nPtr->prev = dl->head.prev;
nPtr->next = &(dl->head);
```

마지막 노드와 head 사이에 새로운 노드를 연결하도록 구현하였다.

---

### 특정 위치 삽입 기능
사용자가 입력한 위치(index)에 새로운 노드를 삽입할 수 있도록 `insertAtDLinkedList()` 함수를 구현하였다.

```c
for (int i = 0; i < index; i++) {
    cur = cur->next;
}
```

지정한 위치까지 이동한 후 이전 노드와 현재 노드 사이에 새로운 노드를 연결하였다.

---

### 맨 앞 삭제 기능
`deleteFirstDLinkedList()` 함수를 이용하여 리스트의 첫 번째 노드를 삭제하도록 구현하였다.

```c
temp->prev->next = temp->next;
temp->next->prev = temp->prev;
```

삭제 후 이전 노드와 다음 노드를 다시 연결하도록 구현하였다.

---

### 특정 위치 삭제 기능
`deleteAtDLinkedList()` 함수를 이용하여 특정 위치의 노드를 삭제하도록 구현하였다.

```c
if (index < 0 || index >= dl->size)
```

잘못된 위치 입력 시 빈 값을 반환하도록 구현하였다.

삭제 후에는 메모리를 해제하고 리스트 크기를 감소시키도록 구현하였다.

```c
free(cur);
dl->size--;
```

---

### 특정 값 삭제 기능
`deleteItemDLinkedList()` 함수를 추가 구현하여 입력한 x, y 값과 일치하는 노드를 삭제하도록 구현하였다.

```c
if (cur->x == x && cur->y == y)
```

리스트를 순회하면서 동일한 값을 가진 노드를 탐색 후 삭제하였다.

---

### 전체 목록 출력 기능
현재 이중 원형 연결 리스트에 저장된 모든 노드를 출력하도록 구현하였다.

```c
printf("[%d] x: %d, y: %d\n",
```

각 노드의 index, x값, y값을 출력하도록 구현하였다.

---

### 리스트 크기 확인 기능
현재 리스트에 저장된 노드 개수를 출력하도록 구현하였다.

```c
printf("현재 크기: %d\n", myList->size);
```

---

### 프로그램 종료 기능
프로그램 종료 시 `destoryDLinkedList()` 함수를 호출하여 모든 동적 메모리를 해제하도록 구현하였다.

```c
destoryDLinkedList(myList);
```
