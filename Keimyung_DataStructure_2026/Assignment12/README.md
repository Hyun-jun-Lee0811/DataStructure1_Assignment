# [자료구조1] Assignment 12 - 2026.05.09

## 과제
업로드된 연결리스트(Linked List) 코드의 delete 함수 오류를 수정한다.
- 연결리스트의 삭제 연산이 정상적으로 수행되도록 구현한다.
- 기존 코드 구조를 유지하면서 오류를 수정한다.

---

## 수정 내용

### 문제점
기존 `deleteAtLinkedList()` 함수에서는 삭제할 노드의 이전 노드를 잘못 탐색하고 있었다.
삭제 대상 노드가 아닌 다음 노드가 삭제되는 문제가 발생하였다.
또한 노드 삭제 후 리스트 크기(size)가 감소하지 않는 문제도 존재하였다.
```c
for (int i = 0; i < at; i++)
```

### 수정
- 삭제할 노드의 이전 위치까지 이동하도록 반복 조건을 수정하였다.
- size 감소를 추가하여 삭제 후 리스트 크기가 정상적으로 유지되도록 수정하였다.
```c
LinkedList:
[0] x:10, y:20
[1] x:20, y:30
[2] x:30, y:40

deleteAtLinkedList(myList1, 1)

LinkedList:
[0] x:10, y:20
[1] x:30, y:40
```
