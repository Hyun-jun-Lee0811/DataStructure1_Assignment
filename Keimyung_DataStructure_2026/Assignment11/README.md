# [자료구조1] Assignment 10-2 - 2026.04.16
##  과제
- 두 개의 다항식을 입력받아 순차 리스트에 저장하고, 순차 리스트 간의 연산을 통해 그 곱셈한 결과를 출력한다.

  - 데이터의 저장은 반드시 순차 리스트를 사용해야 하며, 순차리스트 할당 구조는 자유롭게 선택할 수 있다.

- 다항식의 입력은 지수와 계수 사이에 X^N이 들어가고 더하기 연산자로 연결된다(X의 0차항도 동일하게 적용한다).

  - 다항식 입력 예)  3X^2+2X^1+1X^0  // 3x²+2x+1이라는 뜻

- 수업시간에 한 덧셈코드의 오류도 수정해서 같이 등록한다.
  
### 내용
- #### 기존 덧셈 알고리즘 오류 수정
  - while문 종료 직후, 포인터(i, j)가 리스트의 끝에 도달하지 않은 다항식의 잔여 항들을 addItemPolynomial 함수를 통해 결과 리스트에 강제 병합하는 루프를 추가함
```c
for (; i < sizeArrayList(p1); i++) {
    addItemPolynomial(r, p1->data[i]);
}
for (; j < sizeArrayList(p2); j++) {
    addItemPolynomial(r, p2->data[j]);
}
```
  - 다항식 곱셈 연산
    - 두 다항식의 모든 항 조합을 상호 곱함
    - 계수는 두 항의 곱으로 계산하고, 지수는 지수 법칙에 따라 합산함.
    - `addItemPolynomial` 내부의 지수 비교 로직을 활용하여, 곱셈 결과로 생성된 항이 기존 리스트에 이미 존재하는 지수일 경우 계수를 합산하고, 새로운 지수일 경우 내림차순 정렬 위치에 삽입함.
### 실행 예시
```c
다항식 1 입력 (계수 지수): 2 1
0 -1
다항식 2 입력 (계수 지수): 2 3
0 -1

--- 결과 ---
Polynomial: 2X^1
Polynomial: 2X^3
덧셈 결과: Polynomial: 2X^3+2X^1
곱셈 결과: Polynomial: 4X^4
```
