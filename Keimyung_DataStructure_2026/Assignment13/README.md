# [자료구조1] Assignment 13 - 2026.05.14

## 과제
공지사항의 연결 리스트(Linked List) 소스 코드를 바탕으로, 사용자가 메뉴를 통해 데이터를 자유롭게 추가, 삭제 및 출력할 수 있는 프로그램을 구현하였다.

메뉴 기능은 다음과 같다.

1. 맨 앞에 추가  
2. 맨 뒤에 추가  
3. 특정 위치에 추가  
4. 특정 위치 삭제  
5. 전체 목록 출력  
6. 리스트 크기 확인  
0. 종료

---

## 구현 내용

### 메뉴 기반 연결 리스트 프로그램 구현
반복문을 이용하여 사용자가 원하는 기능을 계속 선택할 수 있도록 구현하였다.

```c
while (1)
```

사용자가 메뉴 번호를 입력하면 각 기능이 실행되도록 구성하였다.

---

### 맨 앞 삽입 기능
`insertFirstLinkedList()` 함수를 이용하여 새로운 노드를 리스트의 가장 앞에 추가하였다.

```c
insertFirstLinkedList(list, (PointType) {x, y, 0});
```

---

### 맨 뒤 삽입 기능
`insertLastLinkedList()` 함수를 이용하여 새로운 노드를 리스트의 마지막에 추가하였다.

기존 코드에서 빈 리스트일 경우 프로그램이 종료되는 문제가 있어 이를 수정하였다.

```c
if (li->head == NULL) {
    li->head = nPtr;
}
```

---

### 특정 위치 삽입 기능
사용자가 입력한 위치(index)에 새로운 노드를 삽입할 수 있도록 `insertAtLinkedList()` 함수를 추가 구현하였다.

```c
for (int i = 0; i < at - 1; i++) {
    ptr = ptr->next;
}
```

지정한 위치 이전 노드까지 이동한 뒤 새로운 노드를 연결하였다.

---

### 특정 위치 삭제 기능
`deleteAtLinkedList()` 함수를 이용하여 특정 위치의 노드를 삭제하도록 구현하였다.

잘못된 위치 입력 시 오류 메시지를 출력하도록 하였다.

```c
if (li == NULL || at < 0 || at >= li->size)
```

또한 삭제 후 메모리를 해제하고 리스트 크기를 감소시키도록 구현하였다.

```c
free(fPtr);
li->size--;
```

---

### 전체 목록 출력 기능
현재 연결 리스트에 저장된 모든 노드를 출력하도록 구현하였다.

```c
printf("[%d] x:%d, y:%d, next:%x\n",
```

각 노드의 index, x값, y값 및 next 포인터를 출력하였다.

---

### 리스트 크기 확인 기능
현재 연결 리스트에 저장된 노드 개수를 출력하도록 구현하였다.

```c
printf("리스트 크기 : %d\n", sizeLinkedList(list));
```

---

### 프로그램 종료 기능
프로그램 종료 시 `destroyLinkedList()` 함수를 호출하여 모든 동적 메모리를 해제하도록 구현하였다.

```c
destroyLinkedList(list);
```
