# [자료구조1] Assignment 10-2 - 2026.04.16
##  과제
- 10*10 크기의 행렬을 순차리스트로 형성하고, 이를 0이 아닌 원소 20개가 있는 희소행렬로 초기화한다.
- 이 희소행렬을 기존의 순차리스트를 사용하여 표현 및 저장하고, 이 희소행렬의 전치행렬을 만드는 프로그램을 실행하여 데이터 이동 횟수를 출력한다. 
### 내용
- SparseMatrix 구조체가 arrayList 객체를 포인터로 관리
- SparseMatrix -> arrayList* -> elementArrayList* (data)
- 100칸(10x10)을 모두 할당하는 대신, 0이 아닌 유효한 원소 20개만 리스트에 저장
- 원본 행렬의 열 번호를 0부터 순차적으로 탐색하며, 해당 열에 속하는 원소를 찾아 행과 열을 맞바꾼 뒤 새로운 리스트에 삽입.
- 내부 루프가 실행될 때마다 moveCount를 증가시켜 전체 데이터 탐색 및 비교 횟수를 계산
- MatrixTerm: 행, 열, 값을 저장하는 단위 구조체
- SparseMatrix: 전체 행/열 크기와 원소들을 담고 있는 순차 리스트 포인터를 포함
