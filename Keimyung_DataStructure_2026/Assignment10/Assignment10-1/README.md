# [자료구조1] Assignment 10-1 - 2026.04.16
## 과제 개요
- 깃허브에 등록된 과제(9)의 ArrayList 코드를 바탕으로 양의 정수 전용 순차리스트를 만든다.
- 만든 양의 정수 순차 리스트에 대하여 다음과 같은 메뉴 기능을 추가하고, 이를 사용해 동작하는 프로그램을 작성한다.
  -    1. 양의 정수와 위치를 입력 받아서 리스트에 추가 (insert)
  -    2. 위치를 입력 받아서 리스트에서 삭제 (delete)
  -    3. 양의 정수와 위치를 입력 받아서 항목의 값의 변경 (replace)
  -    4. 양의 정수 리스트을 위치와 함께 프린트 (print)
  -    5. 양의 정수 리스트에서 모든 값을 삭제 (clear)
## 내용
- isFullArrayList가 가득 찼을 때, 기존 용량의 2배로 자동 확장하여 빈번한 메모리 재할당으로 인한 성능 저하를 방지
- 안정적인 메모리 관리: realloc 실패 시 발생할 수 있는 메모리 누수 및 포인터 유실을 막기 위해 **임시 포인터**를 사용하여 할당 여부를 먼저 검증하도록 구현

## 기능
- 추가: 원하는 위치에 양의 정수를 삽입하며, 용량이 부족할 경우 자동 확장합니다.
- 삭제: 특정 위치의 데이터를 삭제하고 리스트를 앞쪽으로 당겨 빈칸을 채웁니다.
- 변경: 특정 위치의 값을 새로운 양의 정수로 변경합니다.
- 출력: 현재 리스트의 요소들과 함께 size(사용 중인 칸) 및 capacity(전체 할당된 칸)를 출력합니다.
- 초기화: 리스트의 모든 내용을 비우고 논리적인 크기를 0으로 만듭니다.
