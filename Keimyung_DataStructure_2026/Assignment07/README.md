# [자료구조1] Assignment 7 - 2026.04.09

## 과제 개요

구조체 배열을 동적 할당으로 생성하고,
파일 입출력을 통해 학생 이름 데이터를 읽어옵니다.

이후 무작위로 학생 정보를 구성하고,
배열 연산과 포인터 연산을 각각 사용하여
최고점, 최저점, 평균 점수를 계산합니다.

---

## 과제1: 구조체 배열 (인덱스 연산자)

### 내용

- 사용자로부터 학생 수 N 입력
- `malloc()`을 이용하여 크기 N인 구조체 배열 생성
- 파일`(name_list.txt)`에서 이름 목록 읽기
- 학생 정보 구성
  - 이름: 파일에서 랜덤 선택
  - 학번: 1 ~ N
  - 성적: 1 ~ 100 랜덤 값
- 모든 배열 접근을 인덱스 연산자 `(arr[i])` 사용
- 최저점, 최고점, 평균 점수 계산

---

## 과제2: 구조체 배열 (포인터 연산자)

### 내용

- 동일한 학생 배열을 대상으로 수행
- 모든 배열 접근을 포인터 연산 `(*(arr + i))` 사용
- 최소/최대/평균 계산 로직 구현
---

### 실행 결과

```text
학생 수: 3

인덱스 연산자
최저점: Maya Powell(1학번) -> 15점
최고점: Charles Price(2학번) -> 65점
평균점수: 33.00

포인터 연산자
최저점: Maya Powell(1학번) -> 15점
최고점: Charles Price(2학번) -> 65점
평균점수: 33.00
```
